# aws-s3-laravel
aws-s3 server in laravel project

###### install this third party library:
```composer require league/flysystem-aws-s3-v3```


###### AWS Configurations: Update this information with your own key
```
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_DEFAULT_REGION=
AWS_BUCKET=
AWS_URL=
```
